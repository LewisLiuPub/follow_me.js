# follow_me.js

A person following robot demo by using `node-realsense` and DFRobot HCR JavaScript API.

## Bootstrap
```
$ npm install
```

## Run

```
$ node main.js
```

Open web browser and navigate to the `http://Device_IP:8000/index.html`.
Press the `FOLLOW ME` to start.
